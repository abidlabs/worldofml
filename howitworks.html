How It Works

1. How Do I Use This Site?

We've put together a collection of the best videos available on YouTube to help you understand a variety of machine learning (ML) topics. Simply search for the concept you're interested understanding better and you'll be shown the video with a short description.

2. Do I Need to Know the Basics of ML?

Each video includes a prerequisite graph that lists exactly what topics you need to know first in order to understand the video. Don't know one of the prerequisites? Simply click on it to go to that video directly. Postrequisites are also provided to help you figure out where to go next!

3. Where Do I Even Start?

A good place to start may be to click on to see all of our videos. Using a breadth-first search, we list the topics in order from basic concepts to more advanced topics. Every time you watch a video, we store that information (on your computer -- it's all private!) so you can keep track of which concepts you've covered. Alternatively, check out our total to see which concepts are important.

4. How Can I Help?

This entire website is a few static pages hosted on Github. Our collection of videos and our prerequisite graph is a single javascript file (data.js) that can be edited by anyone with a Github account in a transparent manner. Simply fork the Github repository, add a video (make sure to include a title and video URL -- and if you'd like, a description of the video) or a prerequisite, and submit a pull request. Your changes will go live soon!
